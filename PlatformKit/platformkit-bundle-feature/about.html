About

PlatformKit is a platform modelling and platform dependency management toolkit. It can be used to support development processes based on the Model Driven Architecture. PlatformKit is a refactored version of CDDToolkit.

A case study on how to use PlatformKit can be found at http://soft.vub.ac.be/soft/research/mdd/casestudies.

See also the release notes.
