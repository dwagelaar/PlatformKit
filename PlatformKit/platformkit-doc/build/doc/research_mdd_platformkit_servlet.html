Introduction

The PlatformKit Servlet is a web-based interface for picking the right software download for a given platform (Platform-Driven Deployment). It uses the Platformkit Model of all product configurations and the pre-classified ontology generated by the PlatformKit Eclipse Plugin as a basis for reasoning. The servlet redirects the client to the URL that provides the right software download (or ‘none/’ if no download is valid). This URL is based on the URL of the Platformkit Model and the ID of the chosen Constraint Set within that model (e.g. ‘http://ssel.vub.ac.be/platformkit/instantmessenger/’ + ‘default/applet/’).

Download

An example installation of the servlet is running on this website: http://ssel.vub.ac.be/platformkit/. No binary download is provided, since one needs to configure the deployment descriptor (web.xml) for secure use (only trusted sites should be allowed to provide the Platformkit Model). The source files can be found here.
