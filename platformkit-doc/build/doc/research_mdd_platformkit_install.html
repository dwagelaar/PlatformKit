Installing the PlatformKit Eclipse Plugin

  • Back to the PlatformKit main page

System Requirements

You’ll need the following installed on your computer:

  • Eclipse 3.2
  • Eclipse EMF 2.2
  • Eclipse UML2 2.0
    (Update site: http://download.eclipse.org/tools/uml2/updates/site.xml)
  • ATL Development Tools
    (Update site: http://ssel.vub.ac.be/tools/atl/update/)
  • Jar2UML (Optional)
    (Update site: http://ssel.vub.ac.be/tools/jar2uml/update/)
  • Protégé (for ontology editing)
  • A DIG reasoner, such as Racer, Pellet or Fact.

More help on how to install Eclipse features from update sites can be found here.

The example used in the documentation is based on the Instant Messenger case study. You’ll need the following extras installed on your computer to run the examples:

  • Subclipse
    (Update site: http://subclipse.tigris.org/update_1.2.x/)
  • ATLAS MegaModel Management
    (Update site: http://ssel.vub.ac.be/tools/atl/update/)
  • Topcased UML2 editor
    (Update site: http://topcased-mm.gforge.enseeiht.fr/release/update-site/)
  • UML2 Case Studies infrastructure plug-in
    (Update site: http://ssel.vub.ac.be/tools/uml2cs/update/)

You also need to follow the instructions given on the UML 2 Case Studies Installation Instructions page, starting at “Downloading from Subversion”.

Download

To download the PlatformKit Eclipse plugin, do the following in your Eclipse:

  • Select “Help → Software Updates → Find and Install...” from the main menu.
  • Select “Search for new features to install” and click “Next”.
  • Click “New Remote Site...”.
  • Choose a name, fill in http://ssel.vub.ac.be/platformkit/update as URL and click “OK”.
  • Click “Finish”.
  • Select the newly created update site and click “Next”.
  • Now proceed with the installation as you normally would.
